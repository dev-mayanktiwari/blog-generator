# UI Package

This package contains reusable React components based on shadcn/ui, used throughout the Blog Generator monorepo.

## Components
- Button
- Card
- Input
- Label
- Select
- Switch
- Sonner (toast)

## Usage
Import components in your app:
```tsx
import { Button, Card } from '@workspace/ui';
```

## Customization
All components are theme-aware and can be styled with Tailwind CSS.
